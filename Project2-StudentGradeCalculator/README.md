# Java Project 2 – Student Grade Calculator

## Description
In this project, I developed a functional application capable of handling multiple subject inputs, calculating total marks, computing average percentages, assigning grades automatically, determining Pass/Fail status, and validating user input for accurate data processing.

This project is a Java-based Student Grade Calculator developed as part of my DecodeLabs Industrial Internship journey.

The application allows users to enter marks for multiple subjects and automatically calculates:

* Total Marks
* Average Percentage
* Grade Assignment
* Pass/Fail Status

This project demonstrates practical implementation of arithmetic operations, loops, conditional logic, and structured data processing.

## Features

* Multiple subject marks input
* Total marks calculation
* Percentage calculation
* Automatic grade generation
* Pass/Fail indicator
* Input validation
* User-friendly console interface

## Technologies Used

* Java
* VS Code
* JDK 17+
