# Java Project 3 – ATM Interface

## Description

This project is a Java-based ATM Interface system developed as part of my DecodeLabs Industrial Internship journey.

The application simulates real-world ATM operations using Object-Oriented Programming principles and includes secure banking functionalities such as:

* PIN Authentication
* Deposit
* Withdrawal
* Balance Inquiry
* Transaction History
* Input Validation

This project demonstrates practical implementation of secure transaction handling, encapsulation, classes, objects, and real-world banking logic.

## Features

* Secure PIN login system
* Deposit money
* Withdraw money
* Balance checking
* Transaction history tracking
* Input validation
* Insufficient balance protection
* Menu-driven console interface
* OOP architecture

## Technologies Used

* Java
* VS Code
* JDK 17+
